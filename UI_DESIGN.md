## 🎨 Màu sắc & Thiết kế

### Bảng màu
- **Background**: #f5f5f5 (xám nhạt)
- **Header**: #fff (trắng)
- **Text chính**: #2c3e50 (xanh đen)
- **Text phụ**: #7f8c8d (xám)
- **Primary button**: #3498db (xanh dương - Đăng nhập)
- **Danger button**: #e74c3c (đỏ - Đăng xuất)
- **Success button**: #27ae60 (xanh lá - Thêm vào giỏ)

### Điểm nổi bật
- Card sản phẩm có shadow khi hover
- Button có transition mềm mại
- Layout responsive (grid auto-fit)
- User info box với background nhạt

## 💬 User Interactions

### Chưa đăng nhập:
1. User nhấp "Đăng nhập"
2. Modal hiển thị 5 tùy chọn
3. User chọn 1 trong 5 provider
4. Xác thực qua provider
5. Quay lại trang chủ, hiển thị thông tin

### Đã đăng nhập:
1. User nhấp "🛒 Thêm vào giỏ"
2. Alert xác nhận: "Đã thêm ... vào giỏ hàng!"
3. (Tuỳ chọn) Quản lý giỏ hàng

### Đăng xuất:
1. User nhấp "Đăng xuất"
2. Xóa localStorage
3. Quay về trạng thái chưa đăng nhập
